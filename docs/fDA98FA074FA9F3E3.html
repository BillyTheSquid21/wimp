Description

Waits for a specified I/O @a condition on @a socket.

Waits until @a condition will be met on @a socket or an error occurred. If timeout was set using p_socket_set_timeout() and a network I/O operation doesn't finish until timeout expired, call will fail with #P_ERROR_IO_TIMED_OUT error code.
